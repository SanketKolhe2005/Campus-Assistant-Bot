# Campus Assistant Bot

## Project Overview
Campus Assistant Bot is an AI-powered chatbot developed using Dialogflow to help students get instant information about college services.

## SDG
SDG 4 – Quality Education

## Features
- Admission Process Information
- College Timings
- Fee Details
- Library Timings
- Contact Information
- Fallback Response Handling

## Technology Used
- Dialogflow
- NLP
- Google Cloud
